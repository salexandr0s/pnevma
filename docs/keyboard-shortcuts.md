# Keyboard Shortcuts

## Command Palette

- `command_palette.toggle`: `Mod+K`
- `command_palette.next`: `ArrowDown`
- `command_palette.prev`: `ArrowUp`
- `command_palette.execute`: `Enter`

## Pane Navigation

- `pane.focus_next`: `Mod+]`
- `pane.focus_prev`: `Mod+[`

## Quick Actions

- `task.new`: `Mod+Shift+N`
- `task.dispatch_next_ready`: `Mod+Shift+D`
- `review.approve_next`: `Mod+Shift+A`

## Keyboard-first flow

Primary workflows are reachable within two actions:

1. Open palette (`Mod+K`)
2. Type command name/prefix and press `Enter`

## Customization

Edit shortcuts in Settings pane.

Keybindings are persisted to:

- `~/.config/pnevma/config.toml`

Keybindings are applied at runtime without restarting the app.
Duplicate shortcuts are flagged in Settings before saving.
